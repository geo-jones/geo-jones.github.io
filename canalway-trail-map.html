Cycle the Canalway Trail Map

Interactive map for promotion of the Erie Canalway Trail, including tools for planning a trip and cycling-specific amenities. Built with ArcGIS Experience Builder. Includes responsive design optimized for mobile devices.

Project also included creation of embeddable maps on specific topics (e.g. trail closures, camping, bike friendly businesses) which draw from the same underlying web map to simplify map maintenance.
